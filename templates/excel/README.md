# Struktur File Excel / Import Google Sheets

Folder ini berisi template **CSV per sheet** sesuai struktur MVP.

## Cara pakai cepat

1. Buka file `.csv` di Excel.
2. Simpan sebagai workbook `.xlsx` jika ingin digabung jadi 1 file.
3. Import ke Google Sheets sesuai nama tab masing-masing.
